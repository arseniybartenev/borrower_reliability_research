## Описание проекта

Проект выполнен для кредитного отдела банка. На основе данных о заёмщиках необходимо выявить факторы, которые влияют на возврат кредита в срок. Результаты помогут построить модель кредитного скоринга и оценивать риски при выдаче займов.

**Цель:** Провести анализ данных и определить, как различные характеристики заёмщиков (количество детей, семейное положение, уровень дохода, цель кредита) связаны с фактом возникновения задолженности.

## Данные

Исходные данные содержатся в файле `data.csv`. В датасете представлена информация о заёмщиках:

- `children` — количество детей
- `days_employed` — трудовой стаж (в днях)
- `dob_years` — возраст (в годах)
- `education` — уровень образования
- `education_id` — идентификатор образования
- `family_status` — семейное положение
- `family_status_id` — идентификатор семейного положения
- `gender` — пол
- `income_type` — тип занятости
- `debt` — наличие задолженности (1 — есть, 0 — нет)
- `total_income` — ежемесячный доход
- `purpose` — цель получения кредита

## Этапы работы

1. **Предобработка данных**
   - Заполнение пропусков в столбцах `days_employed` и `total_income` медианными значениями по группам `income_type`.
   - Преобразование отрицательных значений стажа в положительные.
   - Удаление строк с аномальным количеством детей (`-1` и `20`).
   - Приведение записей об образовании к нижнему регистру для устранения неявных дубликатов.
   - Удаление полных дубликатов строк.
   - Изменение типа данных `total_income` на целочисленный.

2. **Категоризация данных**
   - Создание столбца `total_income_category` с категориями дохода: A, B, C, D, E.
   - Создание столбца `purpose_category` с обобщёнными целями кредита: операции с автомобилем, с недвижимостью, проведение свадьбы, получение образования.

3. **Исследование зависимостей**
   - Анализ влияния количества детей на возврат кредита.
   - Анализ влияния семейного положения.
   - Анализ влияния уровня дохода.
   - Анализ влияния цели кредита.

4. **Формулирование выводов и рекомендаций**

**Основные выводы:**
- С ростом количества детей (от 0 до 2) увеличивается доля просрочек.
- Самые надёжные заёмщики — вдовцы/вдовы и разведённые; самые рискованные — не состоящие в браке и живущие в гражданском браке.
- Низкий доход (категория E) и средний доход (категория C) связаны с повышенным риском невозврата.
- Кредиты на образование и покупку автомобиля чаще приводят к задолженности, чем ипотечные или свадебные займы.

## Выводы

На основе проведённого анализа можно выделить портрет рискованного заёмщика: наличие двух и более детей, отсутствие официального брака, низкий доход, цель кредита — образование или автомобиль. Рекомендуется учитывать эти факторы при скоринге, чтобы минимизировать невозвраты.
